3. Specific Requirements

3.1 External interfaces

3.1.1 Amazon AWS Database

An Amazon AWS Database service shall be used to store user data from the application.

  • 3.1.1.1 The database shall store consumer user accounts, commercial user accounts, admin user accounts, events, group events, and related data.
  • 3.1.1.2 Input to the database shall come from consumer users and commercial users.
  • 3.1.1.3 Output from the database shall be displayed to consumer users and commercial users in multiple areas of the application.
  • 3.1.1.4 The database shall store information about connected consumer user social media accounts [3.1.3-3.1.5].
  • 3.1.1.5 The database shall store information about monthly bidding and commercial user payment histories [3.1.6].

3.1.2 Google Maps API/GPS interface

Google Maps API will be used to map the available local events and group events posted by commercial users and consumer users.

  • 3.1.2.1 The map feature shall allow consumer users to view local events and group events.
  • 3.1.2.2 The map feature shall allow consumer users and commercial users to add event and group event locations.
  • 3.1.2.3 The map feature shall allow consumer users to view their distance from event and group event locations.
  • 3.1.2.4 The map feature shall allow consumer users to view directions to event and group event locations.

3.1.3 Facebook

This section outlines the requirements that will need to be followed to implement Facebook’s API according to the standards of the stakeholders allowing users to share activities on the social platform.

  • 3.1.3.1 Registered/anonymous users shall have the availability to share events on the Facebook platform if logged into the user’s portal.
  • 3.1.3.2 Stakeholders/vendors shall have access to information being shared about their business pertaining to shared activities.
  • 3.1.3.3 An event shall contain a picture of the product or a logo of the vendor.
  • 3.1.3.4 A deal shall contain the time until expiration.
  • 3.1.3.5 Shared activities that do not include a vendor shall have the availability to upload an image.

3.1.6 PayPal and Google Pay

PayPal Developer and Google Pay APIs shall be used by commercial users to place bids.

  • 3.1.6.1 Both APIs shall allow businesses to place their bids each month.
  • 3.1.6.2 Upon request, commercial users shall receive a report of bidding history.
